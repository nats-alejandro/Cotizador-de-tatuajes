# Tattoo Price Calculator – UX/UI Case Study

Proyecto UX/UI enfocado en diseñar una herramienta digital que permita estimar el costo de un tatuaje de forma rápida según sus características.

## Contexto

En muchos estudios de tatuaje, gran parte del tiempo se destina a responder consultas repetitivas sobre precios.

Los clientes suelen preguntar:

- ¿Cuánto cuesta un tatuaje?
- ¿Depende del tamaño?
- ¿Cuántas sesiones necesita?

Este proceso manual consume tiempo y dificulta la gestión de clientes.

## Problema

Los estudios de tatuaje necesitan una forma rápida y clara de entregar estimaciones de precio sin tener que responder manualmente cada consulta.

## Objetivo

Diseñar una herramienta que permita:

- Estimar precios de tatuajes
- Reducir consultas repetitivas
- Entregar información clara al cliente
- Facilitar el proceso de reserva

## Solución

Se diseñó un **cotizador de tatuajes interactivo** que calcula un rango estimado de precio según:

- Tamaño del tatuaje
- Estilo
- Ubicación en el cuerpo
- Tiempo estimado de sesión

## User Flow

El flujo principal del usuario es:

1. Seleccionar tamaño del tatuaje
2. Elegir estilo
3. Seleccionar ubicación en el cuerpo
4. Revisar precio estimado

## Wireframes

Se desarrollaron wireframes para definir la estructura de la interfaz y la jerarquía de información.

## Diseño de Interfaz

El diseño final busca:

- Claridad en la información
- Proceso paso a paso
- Interacción simple
- Lectura rápida del precio estimado
